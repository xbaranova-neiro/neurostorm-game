# Нейрошторм — MVP (мобильный web)

Вертикальная мини-игра по ТЗ: **70 с** смены, **3 шкалы** (деньги / время / энергия), объекты (польза / ловушки / бустеры), **развилки**, **4 архетипа** финала + запасной, универсальный и персонализированный CTA.

## Запуск локально

Из папки `neurostorm`:

```bash
cd neurostorm
python3 -m http.server 8080
```

Откройте в браузере: `http://localhost:8080`  
Для мобильного теста — с телефона в той же Wi‑Fi сети: `http://<ваш-IP>:8080`.

**Важно:** в `index.html` подключён **`js/bundle.js`** (сборка без ES‑модулей), чтобы игра работала и при **двойном клике по `index.html`** (`file://`). Если меняете файлы в `js/*.js`, пересоберите бандл:

```bash
cd neurostorm
npx esbuild js/main.js --bundle --outfile=js/bundle.js --format=iife --platform=browser
```

## Управление

- **Тап** по зелёным/фиолетовым карточкам — поймать пользу или бустер.
- **Свайп в сторону** по красной ловушке — сбросить; если уйдёт вниз или тапнуть — штраф.
- **Развилки** — модальное окно, игра на паузе.

## Аналитика

События пишутся в консоль (`[analytics] ...`) и дублируются событием `neurostorm_analytics` на `window`. Подключите свой трекер в `js/analytics.js`.

События: `game_start`, `role_selected`, `wave_completed`, `booster_caught`, `trap_hit`, `decision_choice`, `final_archetype`, `cta_click`.

## Структура

- `index.html` — экраны
- `css/main.css` — стили, контейнер ~9:16
- `js/data.js` — **контент**: роли (питч + история), объекты с **хуками** (подпись смысла), развилки (**контекст + подсказки** к вариантам), архетипы (боль, инсайт, ошибки, кейс, шаг, CTA)
- `js/narrative.js` — **голос рынка**: тикер в HUD, темы волн, варианты хаоса, оттенки комбо
- `js/game.js` — движок
- `js/diagnostics.js` — выбор архетипа
- `js/main.js` — навигация
- `js/analytics.js` — заглушка трекинга
